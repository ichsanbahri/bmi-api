What is BMI API

BMI API is Golang based project for ideal weight calculator application. why choose Golang, because Golang is very good in building large, easy, fast and safe APIs. This is my first project using golang
You can use it with GitHub as well as a standalone project.

What is Golang?

Go is an open source programming language that makes it easy to build simple, reliable, and efficient software.

Full Go documentation You can find full Go Lang documentation here.

Install Go

Installing Go should be straight-forward if all requirements are met. Before you start, make sure your system has the following:

  • GNU/Linux, Windows, or macOS
  • Go version 1.17 or above, including all development headers
  • Go-Chi chi is a lightweight, idiomatic and composable router for building Go HTTP services

Install Go Linux

The best way to install go is via Go. At the terminal prompt, simply run the following command to install go:

1.Extract the archive you downloaded into /usr/local, creating a Go tree in /usr/local/go.

ImportantThis step will remove a previous installation at /usr/local/go, if any, prior to extracting. Please back up any data before proceeding. 

$ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.17.6.linux-amd64.tar.gz

2.Add /usr/local/go/bin to the PATH environment variable. 

$ export PATH=$PATH:/usr/local/go/bin

3.Verify that you've installed Go by opening a command prompt and typing the following command

$  go version

All of go’s gem dependencies are automatically installed by the above command, so you won’t have to worry about them at all.

Full go installation guide You can find full go installation guide here.

Install BMI-API

Download and Usage

Download your BMI-API from Github (you should get V.1.0.1.zip) and unzip it.

Start BMI-API

go comes with a built-in development server that will allow you to preview what the generated site will look like in your browser locally.

You can run this commands inside theme folder:

$ go run ./api.go
# A BMI-API will run at http://127.0.0.1:8181/

Download BMI-API using git

$ git clone https://github.com/ichsanbahri/bmi-api.git
  
$ cd bmi-api

$ go run api.go

Usage BMI-API

$ api/?height=[insert your height]&weight=[insert your weight]
Atribute Type Description
Weight integer Weight Body
Height integer Height Body
